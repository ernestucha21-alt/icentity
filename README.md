# ICENTITY

Proyecto académico de diseño y desarrollo web.

## Integrantes
- Ernestina Spollansky
- Zoe Salgado

## Descripción
ICENTITY es una experiencia interactiva que relaciona sabores de helado con distintos perfiles de personalidad, permitiendo al usuario descubrir qué helado lo representa a través de una interfaz dinámica e intuitiva.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap
